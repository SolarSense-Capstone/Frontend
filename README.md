# SolarSense Frontend
Overview

SolarSense AI is a web application that helps businesses evaluate whether switching to solar energy is viable. The platform collects business energy data, sends it to a backend AI-powered analysis system, and returns clear insights on solar feasibility, energy savings potential, and sustainability impact.
This project focuses on improving energy accessibility and affordability, especially in regions with unreliable electricity supply.

# Problem Statement

Many businesses, particularly in developing regions, struggle with unreliable power supply and high electricity costs. However, determining whether solar energy is a viable alternative is often complex, technical, and expensive.
SolarSense AI simplifies this decision-making process using data analysis and AI-driven insights.

# Key Features

Business energy assessment questionnaire

Solar viability analysis via backend AI models

Clear result dashboard with recommendations

Responsive, mobile-friendly interface

API-driven architecture for scalability.

# System Architecture

Frontend → Backend API → Database → AI Models → Backend → Frontend

Frontend collects structured user data

Backend processes requests and communicates with AI models

AI models analyze solar viability

Results returned to frontend for visualization.
